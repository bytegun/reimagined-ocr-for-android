This app is a mash-up of several open source projects:

  • Tesseract OCR
  • Tesseract Tools for Android
  • the ZXing barcode image processing library
  • a San Francisco State University grad school project

Code from other open source projects has also been used, thanks to:

  • google-api-translate-java
  • microsoft-translator-java-api
  • jtar

The source code for this project is available at https://github.com/rmtheis/android-ocr.
